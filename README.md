# How to start?
## Installing
- Clone repository
```
git clone github.com/xairaven/Algorithms-Data-Structures
```
### Enviroment
- C++ :
  - Install C++ IDE (CLion preferably) and CMake
  - Add CMake list of structure to configurations
- Java :
  - Install Java IDE (IntelliJ Idea preferably) and JDK 17.0
